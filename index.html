ViLMN: A Large-scale Dataset for Quadrupedal Robot Vision-Language Manipulation and Navigation

Yukun Wu, Lijun, Zhang Min, Pengxiang Ding,
Han Zhao, Zhiming Zhang, Zhitao Wang, Zhenyu Wei, Donglin Wang
Video Code Data Arxiv

Abstract

In recent years, quadruped robots have emerged as compelling alternatives to traditional robotic systems and have become increasingly active across various fields, such as navigation, express delivery, food delivery and so on. Transformer-based models pretrained on large-scale datasets have demonstrated significant success in numerous domains, however, research progresses have encountered a deceleration in the context of quadruped robots. The primary reason for this phenomenon is the absence of a comprehensive and diverse quadruped dataset. To expedite the integration of quadrupeds into real-world scenarios, in this paper, we present a new benchmark for quadrupedal robot Vision-Language Manipulation and Navigation (ViLMN), which is collected from the real quadruped robot involving indoor and outdoor scenes. Specifically, ViLMN contains 20K trajectories spanning more than 43 diverse tasks. Each trajectory contains task-specific languages and rich multi-modal data, incorporating visual inputs, depth information, robot state parameters and various action sequences. Action sequential information encompasses not only low-level joint positions but also high-level joystick commands provided by a human presenter. We open-source all codes of our benchmark and hope that the proposed ViLMN serves as a valuable resource to facilitate future research on manipulation and navigation tasks for quadruped robots.

ViLMN collected information on the quadruped robot platform, including RGB cameras, depth cameras, LiDAR, IMU, and foot joint information. We provide ROSBAG that covers this information for everyone to use, and each useful data segment has been segmented. In addition, if it is necessary to use our baseline model for reproduction, we have provided our own data processing methods. The detailed usage methods are available in the GitHub repository.

ViLMN Architecture

ViLMN has up to 43 tasks, which can be divided into navigation tasks and manipulation tasks. In addition, based on different interfering factors, we have expanded 43 tasks to 69 scenarios, which will be beneficial for the training of various robot models.

Learning from ViLMN

To evaluate the efficiency of the ViLMN dataset, we propose a baseline model, CLIPoint, which integrates the robust semantic understanding of the CLIP model with global information derived from LiDAR. To align the point cloud with language prompts and RGB observation, we project the 3D point cloud data onto a 2D BEV map, where depth is represented without color information.
